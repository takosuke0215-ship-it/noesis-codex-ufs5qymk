# NOESIS

このフォルダーは、Edgeで開かれていたiCloudメモとChatGPT対話ログの可視情報から抽出したNOESIS資料庫です。

## 情報の優先順位

ユーザー確認により、NOESIS設定は「ChatGPTとの最新の対話ログ」を正とします。

1. 最新ChatGPT対話ログ: 正式設定、現在の方針
2. iCloudメモ本文: 本文素材、旧稿、下書き、比較用
3. 抽出時の画面断片: 補助メモ

## 構成

- `raw/icloud-notes/`: iCloudメモから本文コピーで抽出できた原文メモ
- `raw/chatgpt-visible-extract.md`: ChatGPTログの可視範囲から拾えた設定断片
- `wiki/Home.md`: NOESIS公式Wikiの入口
- `wiki/episodes/`: 話数・本文整理
- `wiki/characters/`: キャラクター設定
- `wiki/settings/`: 世界観、NOESIS、神器、章構成
- `wiki/music/`: テーマ曲・歌詞・MV管理用
- `extraction_report.md`: 抽出結果と未確認事項

## 注意

iCloudメモは本文として13件抽出できました。ただし検索結果リスト経由の自動抽出のため、一部ファイルでタイトルと本文内容がずれている可能性があります。rawは消さず、wiki側では最新ChatGPTログを優先して整理しています。
